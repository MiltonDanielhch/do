# Roadmap de Desarrollo - Transporte

## Funcionalidades Actuales

### Módulo de Asociados
*   Registro de asociados con datos completos
*   Estado activo/inactivo
*   Vinculación con organizaciones
*   Generación de código QR

### Módulo de Vehículos
*   Registro de vehículos por asociado
*   Control de placa única
*   Historial de vehículos

### Módulo de Rutas
*   Gestión de rutas básicas
*   Asignación a organizaciones

### Módulo de Membresías
*   Control de vigencia por asociado
*   Estados: Vigente, Vencido, Cancelado

### Portal Público
*   Búsqueda por CI o placa vehicular
*   Estadísticas generales

*   **Logs de auditoría** - Registro de cambios en datos sensibles
