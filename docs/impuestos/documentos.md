# Documentos y Archivos

## Vista General

Gestión de documentos asociados a los trámites de transmisión.

## Cómo Acceder

**Ruta:** `admin/tramites/{tramite}/documentos`

## Funcionalidades

### 1. Agregar Documento

| Campo | Descripción |
|-------|-------------|
| Nombre | Nombre del documento |
| Archivo | Archivo PDF/Digital |

### 2. Listar Documentos

Ver todos los documentos del trámite.

### 3. Eliminar Documento

Quitar documentos asociados.

---

## Plantillas de Impresión

El sistema incluye múltiples plantillas para imprimir documentos:

### 1. Template Print
Plantilla estándar vertical.

### 2. Template Print Letter
Plantilla en formato carta.

### 3. Template Print Legal
Plantilla formato legal.

### 4. Template Print Horizontal
Plantilla horizontal.

### 5. Template Notification
Plantilla para notificaciones.

## Formato Impresión A01

**Ruta:** `admin/tramites/{tramite}/a01`

Genera el formato oficial A01 del trámite.
