# Reportes del Sistema

## Vista General

El módulo de reportes permite generar consultas e informes consolidados para la toma de decisiones.

## Cómo Acceder

**Ruta:** `admin/reportes`

## Reportes Disponibles

### 1. Reporte de Trámites

Muestra el listado de trámites registrados.

| Campo | Descripción |
|-------|-------------|
| ID | Identificador |
| Fecha | Fecha de registro |
| Estado | Estado del trámite |
| Monto | Impuesto calculado |

### 2. Reporte de Pagos

Informe de pagos realizados por trámite.

### 3. Reporte de Personas

Listado de personas registradas por tipo.

### 4. Reporte de Inmuebles

Estado de inmueb registrados y sus avaluos.

### 5. Reporte de Exenciones

Resumen de exonerraciones aplicadas.

## Consideraciones

- Los reportes pueden filtrarse por fecha
- Soportan exportación a PDF/Excel según tipo
