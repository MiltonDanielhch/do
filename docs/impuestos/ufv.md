# UFV - Valor Unitario

## Vista General

Sistema de gestión de la UFV (Unidad de Fomento de Vivienda) para actualización de valores históricos.

## Cómo Acceder

**Ruta:** `admin/ufvs`

## Funcionalidades

### 1. Listado de UFV

Muestra el histórico de valores UFV.

| Campo | Descripción |
|-------|-------------|
| ID | Identificador |
| Fecha | Fecha del valor |
| Valor | Valor UFV de la fecha |

### 2. Registrar UFV

Agregar nuevos valores UFV.

### 3. Editar UFV

Modificar valores existentes.

## Consideraciones

- La UFV permite actualizar valores históricos a valores actuales
- Se usa en el cálculo del IDTGB para transacciones antiguas
