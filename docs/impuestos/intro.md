# Sistema de Impuestos - Beni

## Gestión Integral de Transmisión de Bienes Gratuitos

¡Te damos la bienvenida a la documentación oficial del Sistema de Impuestos del Gobierno Autónomo Departamental del Beni! Esta guía está diseñada para ayudarte a comprender y utilizar todas las funcionalidades del sistema para la gestión del Impuesto Departamental de Transmisión de Bienes Gratuitos (IDTGB).

El Sistema de Impuestos es una solución integral diseñada para:

- **Automatizar el Cálculo:** Calcula automáticamente el impuesto según tablas vigentes.
- **Centralizar Trámites:** Gestiona todos los trámites de transmisión en una única plataforma.
- **Controlar Exenciones:** Gestiona las exoneraciones de impuesto de forma eficiente.
- **Garantizar Transparencia:** Mantiene trazabilidad completa de cada operación.

## ¿Qué encontrarás en esta documentación?

Esta documentación está estructurada para guiarte a través de los diversos módulos y funcionalidades. Cada sección te proporcionará:

- Una visión general del propósito del módulo.
- Instrucciones paso a paso para realizar tareas específicas.
- Explicaciones claras de conceptos clave.
- Consejos y mejores prácticas.

## Módulos Clave del Sistema

El Sistema de Impuestos se compose de módulos interconectados:

- **[Calculadora IDTGB](/impuestos/calculadora):** Cálculo automático del impuesto departamental.
- **[Wizard de Trámites](/impuestos/wizard-tramites):** Asistente de 7 pasos para registro de trámites.
- **[Personas](/impuestos/personas):** Gestión de disponentes y adquirentes.
- **[Inmuebles](/impuestos/inmuebles):** Control de bienes raíces y avaluos.
- **[Pagos](/impuestos/pagos):** Gestión de pagos asociados a trámites.
- **[Parametros](/impuestos/parametros):** Catálogos del sistema (tasas, tipos, parentescos).

## Beneficios del Sistema

- **Cálculo Automático:** Elimina errores manuales en el cálculo del impuesto.
- **Trazabilidad Completa:** Cada trámite tiene historial completo.
- **Validación en Línea:** Verificación pública de trámites.
- **Integración UFV:** Actualización automática por UFV.
- **Gestión de Exenciones:** Control eficiente de exoneraciones.

## Primeros Pasos

### Calcular un Impuesto
1. Accede a `/calculadora-idtgb-beni`
2. Ingresa los datos requeridos
3. Obtén el cálculo automático

### Registrar un Trámite
1. Accede a Trámites en el panel admin
2. Selecciona "Crear Nuevo" (inicia el wizard)
3. Completa los 7 pasos
4. Confirma y guarda
